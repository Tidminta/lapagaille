Cheatsheet

whoami: permet d'afficher l'user

bin: contient des programmes suscptibles d'etre utilises par tout les user de la machine

boot: fichier permettant le demarage de linux

dev: fichier contenant les peripheriques, ce dossier contient des sous dossiers qui representent chacun un peripherique. On y trouve par exemple le fichier qui represente le lecteur CD

etc: fichiers de configuration editables de plusieurs programmrs ou services du system

home: repertoire perso des users, fichiers perso

lib: dossier contenant les bibliotheques partagees (generalement des fichiers .so) utilises par les programmes

media: lorsqu'un peripherique amovible est insere dans l'ordi Linux permet d'y acceder via ce sous-dossier

opt repertoires utilise pour les ad-ons des prgm

proc: contient des infos systeme Répertoire virtuel pour les informations système (états du noyau et des processus système) 

root: dossier perso de root 

sbin: contient des programmes systeme importants

tmp: dossier temporaire pour stocker des fichiers

usr: (unix system resources) dossiers dans lequel vont s'installer la plupart des programmes demande par l'utilisateur

var: contient des donnees variables, souvent des logs(traces ecrites de ce qui s'est passe recemment sur l'ordi)

opt: Emplacement pour des applications installées hors gestionnaire de paquets (logiciels optionnels) 

run:Informations relatives au système depuis son dernier démarrage (ex : utilisateurs actifs, services en cours d'exécution, etc.) 

srv: Données pour les services du système
